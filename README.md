# 😊 Mood-Based Fact Generator

A fun and simple web app that shows random facts based on your mood.

# Preview
 <img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/0ee255cd-8b3a-495c-bccf-d1823853981f" />

## 🚀 Features

* Select your mood (happy, sad, bored, etc.)
* Get a random fact instantly
* Clean and easy-to-use interface

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Useless Facts API

## ▶️ How to Run

1. Download or clone the repo
2. Open `index.html` in your browser

## 💡 Idea

This project adds a fun twist to random facts by connecting them with your current mood.
